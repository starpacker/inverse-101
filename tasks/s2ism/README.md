# s2ISM: Scalable Structured Image Scanning Microscopy

> Reconstruct a 3-D fluorescent object from multi-channel SPAD detector measurements using multi-plane maximum-likelihood deconvolution, where the challenge is jointly deconvolving 25 detector channels across multiple axial planes with Poisson noise.

> Domain: biology | Keywords: fluorescence microscopy, image scanning microscopy, deconvolution | Difficulty: Medium

## Background

Image Scanning Microscopy (ISM) replaces the single-element detector of a confocal
microscope with a small array detector (e.g., a 5x5 SPAD array). Each detector
element captures a slightly offset view of the sample, encoding both lateral and
axial information. The s2ISM method exploits this multi-channel, multi-plane
structure to achieve super-resolution and optical sectioning beyond what
conventional confocal or simple pixel-reassignment methods can deliver.

## Problem Description

The measurement model relates a 3-D fluorescent object $x$ to multi-channel
detector measurements $y$ through channel- and plane-specific point spread
functions (PSFs):

$$y_c(r) = \sum_{z=1}^{N_z} h_{z,c}(r) * x_z(r) + n_c(r)$$

where:
- $x_z(r)$: fluorophore distribution at axial plane $z$, size $N_y \times N_x$
- $h_{z,c}(r)$: PSF for detector channel $c$ at plane $z$, size $N_y \times N_x$
- $y_c(r)$: measured photon counts at channel $c$, size $N_y \times N_x$
- $n_c(r)$: Poisson noise
- $*$: 2-D spatial convolution
- $N_z$: number of axial planes (2 in this task)
- $c \in \{1, \ldots, 25\}$: SPAD detector channels (5x5 array)

The inverse problem is ill-posed because: (1) the PSF mixes information from
multiple z-planes into each 2-D measurement, (2) the PSF varies across detector
channels, and (3) the measurements are corrupted by Poisson noise.

**Input**: Multi-channel ISM measurement $y$ of shape $(N_y, N_x, N_{ch})$ and
PSF stack $h$ of shape $(N_z, N_y', N_x', N_{ch})$.

**Output**: Reconstructed 3-D object $\hat{x}$ of shape $(N_z, N_y, N_x)$.

## Method Hints

Use a Richardson–Lucy-style maximum-likelihood expectation-maximisation (ML-EM)
algorithm extended for the multi-channel, multi-plane ISM forward model. The key
insight is that the multiplicative ML-EM update naturally handles Poisson noise
statistics and preserves non-negativity.

Perform all convolutions in the Fourier domain for efficiency. The multi-channel
partial convolution can be expressed as an Einstein summation over FFT-domain
tensors, avoiding explicit loops over detector channels.

The PSF is provided directly in this task (simulated from known optical
parameters). In a real-data scenario, it would be estimated from the data via
shift-vector analysis and scalar PSF fitting.
