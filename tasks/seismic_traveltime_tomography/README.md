# Seismic Traveltime Tomography

> Recover a 2D P-wave velocity model from first-arrival traveltimes recorded at surface receivers, given a sparse irregular source distribution and a smooth linear background model.

> Domain: Earth Science | Keywords: wave imaging, adjoint-state method, inverse problems, seismic tomography | Difficulty: Medium

## Background

Seismic traveltime tomography is a fundamental tool in solid-earth geophysics for imaging the subsurface velocity structure of the Earth.  When a seismic source (earthquake or controlled explosion) fires, compressional (P) waves propagate through the Earth and arrive at surface seismographs.  The first-arrival time at each station depends on the velocity field integrated along the fastest path — the raypath — connecting source to receiver.  By collecting many source–receiver pairs with different geometries, one can invert the ensemble of traveltimes to recover the 2D or 3D velocity model.

The technique has broad application across scales: crustal and mantle tomography using regional and teleseismic earthquakes, near-surface site characterization for engineering and hazard assessment, and subsurface exploration for oil, gas, and mineral deposits.  Modern variants of the method (refraction tomography, finite-frequency tomography, ambient-noise tomography) all share the same mathematical core — a PDE-based forward model and a sensitivity kernel computed from its adjoint.

This task implements **adjoint-state traveltime tomography** (ATT), which computes sensitivity kernels analytically from the Eikonal equation and its adjoint rather than by explicit ray tracing in the classical sense.  The formulation is particularly well suited to large-scale 3D problems, but the 2D Cartesian case implemented here captures all the essential physics and numerical challenges.

## Problem Description

**Forward model.** For a given slowness field s(x, z) = 1/v(x, z), the first-arrival traveltime T(x, z) from a point source at (x_s, z_s) satisfies the **Eikonal equation**:

```math
\left(\frac{\partial T}{\partial x}\right)^2 + \left(\frac{\partial T}{\partial z}\right)^2 = s^2(x,z)
```

with the point-source initial condition T(x_s, z_s) = 0.  The measured quantity at receiver position (x_r, z_r) is T(x_r, z_r), evaluated by interpolating the traveltime field at that location.

**Inverse problem.** Given N_src × N_rec observed traveltimes T^obs_{nm} (source n, receiver m), find the slowness model s(x, z) that minimises the least-squares misfit:

```math
\chi(s) = \frac{1}{2} \sum_{n,m} \bigl(T^\text{syn}_{nm}(s) - T^\text{obs}_{nm}\bigr)^2
```

The problem is ill-posed because: (1) the number of free parameters (Nz × Nx grid cells) greatly exceeds the number of measurements (N_src × N_rec traveltimes); (2) the mapping from s to T is nonlinear; and (3) sensitivity is limited to regions sampled by raypaths, leaving unconstrained zones wherever ray coverage is sparse.

**Input:** observed traveltimes `traveltime_obs` (N_src, N_rec), source and receiver positions, and a background velocity model derived from `meta_data.json`.

**Output:** reconstructed 2D velocity field v_inv(z, x) saved as `velocity` (shape (1, Nz, Nx)) and the velocity perturbation relative to the background `velocity_perturbation` (shape (1, Nz, Nx)) in `output/velocity_reconstructed.npz`.

The true velocity model is the Marmousi synthetic model rescaled to a crustal domain (184 km × 60 km, 2 km grid spacing), and traveltimes are computed by solving the Eikonal equation on the true model and adding Gaussian noise (σ = 0.1 s).

## Method Hints

Solve the forward problem for each source using the **Fast Marching Method** (FMM), a single-pass algorithm for the Eikonal equation with O(N log N) complexity.  The Python package `scikit-fmm` provides a direct interface: initialize a signed-distance field `phi` that is negative everywhere except at the source node (set to +1), then call `skfmm.travel_time(phi, speed=1/s, dx=[dz, dx])`.

The gradient of the misfit with respect to slowness equals the **sensitivity kernel** — the integral of the traveltime residual weighted by the path length through each grid cell.  Compute it by **ray back-projection**: trace a ray from each receiver backward to its source along the direction −∇T / |∇T| using Euler integration, and accumulate R_{nm} × Δl at each visited grid node.

Regularize the kernel by **kernel density normalization**: divide by (K_d + ε)^ζ where K_d counts total path length independently of residuals.  Apply an additional Gaussian smoothing step (σ ≈ 1–2 grid cells) to suppress artefacts from sparse coverage.

Update the slowness via gradient descent with a **step-size-controlled** update: choose α so that the maximum fractional slowness change equals a fixed fraction (~2%) of the current maximum slowness, and apply a decay factor (~0.97) per iteration to prevent oscillation as the misfit flattens.
